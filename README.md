# 🍔 Swiggy Data Analysis Project

## 📌 Project Overview
This project focuses on analyzing Swiggy restaurant data using Python, Power BI, and Streamlit.  
The project includes web scraping, data cleaning, exploratory data analysis (EDA), and dashboard visualization to gain meaningful insights into restaurant trends, ratings, cuisines, delivery times, and pricing.

---

## 🚀 Features
- 🌐 Web Scraping using Python
- 📊 Interactive Streamlit Dashboard
- 📈 Data Visualization using Matplotlib & Seaborn
- 🧹 Data Cleaning & Preprocessing
- 📍 Area-wise Restaurant Analysis
- ⭐ Restaurant Ratings Analysis
- 💰 Cost vs Rating Insights
- 🍕 Cuisine Distribution Analysis
- 📊 Power BI Dashboard
